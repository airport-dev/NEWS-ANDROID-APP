# NEWS-ANDROID-APP
このアプリはandroidの勉強用に開発したビジネスニュースアプリです。 NewsApiから取得したデータをListview で表示しまたそこから気になったニュースをurlを通してwebviewで表示しています。

## 機能
* ニュースをリスト表示する機能
* 遷移時のアニメーション
* webViewでのデータ表示機能
